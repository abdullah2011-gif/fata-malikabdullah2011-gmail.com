Admin template scrapped with [Next.js](https://nextjs.org/) project.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

### Technologies Used

- NextJS
- Mui v5
- Tailwind CSS
- SASS
- Framer Motion (animations)
- Redux (react-redux, redux-persist)
- Formik (creating forms)
- Yup (validations)
- Notistack (notifications)
